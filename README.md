# playwright-bdd-example

* == project / run BDD tests -- via -- [playwright-bdd](https://github.com/vitalets/playwright-bdd)  

## how to run locally?

* `npm install`
* `npm run test`
   ```
   Running 2 tests using 1 worker
   2 passed (2.3s)
   ```
